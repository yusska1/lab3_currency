# Лаб №3 — Система синхронизации курсов валют
# Описание проекта

Данный проект реализует серверное приложение для работы с курсами валют.  
Система автоматически загружает данные о курсе валют с сайта Чешского национального банка и сохраняет их в базу данных.

После этого пользователь может получить аналитический отчёт по выбранным валютам за указанный период.


# Функциональность

- Синхронизация данных с внешнего источника  
- Сохранение данных в SQLite базу  
- Получение данных за период времени  
 Формирование отчёта:
- минимальное значение курса
- максимальное значение курса
- среднее значение

REST API:
- /sync — синхронизация данных
- /report — получение отчёта


# Источник данных

Данные берутся с официального сайта:
Czech National Bank (ČNB)  
https://www.cnb.cz




# Запустить сервер
python server.py

открываем в браузере: 
- Синхронизация:
  http://127.0.0.1:5000/sync?start=01.01.2019&end=03.01.2019

- Отчёт:
  http://127.0.0.1:5000/report?start=01.01.2019&end=03.01.2019&currencies=USD,EUR
